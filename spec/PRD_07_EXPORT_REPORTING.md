### 3.3 PDF Specifications

| Margins | 2 cm all sides |
| Font | Helvetica (system) |
| Header Font | 18pt Bold |

### 3.4 Chart Images in PDF

| Property | Specification |
|----------|---------------|
| Source | Canvas toDataURL (PNG) |
| Max Width | 15 cm |
| Max Height | 10 cm |
| Aspect Ratio | Preserved |
| Quality | High resolution (2x scale) |

### 3.5 PDF Generation API

**Endpoint:** POST /export-pdf

**Rate Limit:** 10 requests per minute

**Request Body:**
```json
{
  "results": {
    "age_years": 2.45,
    "weight": { "value": 12.5, "centile": 50.1, "sds": 0.03 },
    "height": { "value": 85.3, "centile": 25.5, "sds": -0.67 },
    // ... full results object
  },
  "patient_info": {
    "sex": "male",
    "birth_date": "2023-06-15",
    "measurement_date": "2025-11-27",
    "reference": "uk-who"
  },
  "chart_images": {
    "height": "data:image/png;base64,iVBORw0KGgo...",
    "weight": "data:image/png;base64,iVBORw0KGgo...",
    "bmi": "data:image/png;base64,iVBORw0KGgo...",
    "ofc": "data:image/png;base64,iVBORw0KGgo..."
  }
}
```

**Response:**
- Content-Type: `application/pdf`
- Content-Disposition: `attachment; filename="growth-report-YYYY-MM-DD-HHMMSS.pdf"`

### 4.3 Conditional Sections

Include sections only if data present:

### 4.4 Copy Implementation

```javascript
async function copyResultsToClipboard() {
    const text = formatResultsAsText(currentResults);

    try {
        await navigator.clipboard.writeText(text);
        showToast('Results copied to clipboard');
    } catch (err) {
        // Fallback for older browsers
        const textarea = document.createElement('textarea');
        textarea.value = text;
        document.body.appendChild(textarea);
        textarea.select();
        document.execCommand('copy');
        document.body.removeChild(textarea);
        showToast('Results copied to clipboard');
    }
}
```

### 4.5 Keyboard Shortcut
- **Ctrl+C** (when results visible) triggers copy
- Conflict handling: Only intercept if focus not on text input

## 5. Chart Download

### 5.1 Purpose
Download current chart as high-quality PNG for:
- Presentations
- Reports created in other software
- Patient education materials

### 5.2 Image Specifications

| Property | Specification |
|----------|---------------|
| Format | PNG |
| Resolution | 2x canvas size (Retina) |
| Background | White (light mode) / Dark grey (dark mode) |
| Filename | `growth-chart-[type]-[date].png` |

### 5.3 Download Implementation

```javascript
function downloadChart() {
    const canvas = document.getElementById('growthChart');
    const chartType = getCurrentChartType();
    const date = new Date().toISOString().split('T')[0];

    // Create high-res canvas
    const scale = 2;
    const exportCanvas = document.createElement('canvas');
    exportCanvas.width = canvas.width * scale;
    exportCanvas.height = canvas.height * scale;

    const ctx = exportCanvas.getContext('2d');
    ctx.scale(scale, scale);
    ctx.drawImage(canvas, 0, 0);

    // Trigger download
    const link = document.createElement('a');
    link.download = `growth-chart-${chartType}-${date}.png`;
    link.href = exportCanvas.toDataURL('image/png');
    link.click();
}
```

### 5.4 Chart Title in Export
Include metadata in exported image:
- Chart type (Height/Weight/BMI/OFC)
- Growth reference
- Generated date

## 8. Error Handling

### 8.1 PDF Generation Errors

| Error | User Message |
|-------|--------------|
| Missing data | "Please calculate results before exporting" |
| Server error | "PDF generation failed. Please try again." |
| Rate limited | "Too many requests. Please wait a moment." |
| Network error | "Network error. Check your connection." |

### 8.2 Copy Errors

| Error | Fallback |
|-------|----------|
| Clipboard API unavailable | Use execCommand('copy') |
| Both fail | Show manual copy instructions |

### 8.3 Chart Download Errors

| Error | Message |
|-------|---------|
| Canvas not available | "Chart not loaded. Please wait." |
| Download blocked | "Download blocked. Check browser settings." |

## 9. Security Considerations

### 9.1 PDF Generation
- Rate limited (10/minute) to prevent abuse
- No user-provided HTML/scripts in PDF
- Chart images validated as base64 PNG
- Server-side generation (not client)

### 9.2 Clipboard
- Only writes text, no HTML
- User-initiated action only
- No automatic clipboard writing

### 9.3 Chart Download
- Client-side only (no server round-trip)
- Standard browser download behavior
